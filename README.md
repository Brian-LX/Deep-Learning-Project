# Deep-Learning-Project
The Inception-ResNet model is applied to multiple classification problems.
The data set used in this project can be downloaded from: https://www.kaggle.com/datasets/dedeikhsandwisaputra/rice-leafs-disease-dataset/.
**Background**: Rice is a staple food for most of the world's population. However, rice is attacked by multiple diseases caused by bacteria, fungi, and viruses, resulting in loss of yield. This project proposes a customized CNN model to provide valuable insights into the classification and health detection of rice leaves.
